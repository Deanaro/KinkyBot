KINKY BOT v1.02

To use this bot, type something then press F10 to add it to a list.
Press F11 to start the bot.
Bot Available at https://github.com/Deanaro/KinkyBot
This bot requires AHK https://autohotkey.com/

F9 - Help
F10 - Add Line
Ctrl+F10 - Reset Array
F11 - Send in loop
Ctrl+F11 - Send
F12 - Pause
Ctrl+F12 - Terminate Bot
