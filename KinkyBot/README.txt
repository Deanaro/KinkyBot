KINKY BOT

To use this bot, type something then press F10, the bot will copy the text and spam it in chat.
You can also edit a variable in this script called linecount to change the amount of lines per message.
Bot available at https://github.com/Deanaro/KinkyBot
This bot requires AHK https://autohotkey.com/

F9 - Help
F10 - Start
F11 - Start without copy
F12 - Pause
Ctrl+F12 - Terminate Bot
